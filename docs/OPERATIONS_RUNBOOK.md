# 运营交接手册

## 运营目标

当前运营目标不是大规模上线，而是完成第一批创作者内测。

核心目标：

```text
找到 30 个创作者 -> 让 10 个创作者完成第一部互动短剧 -> 收集反馈 -> 准备付费测试
```

## 每周运营节奏

周一：

- 招募创作者
- 发布项目进展
- 联系短剧、小说、互动游戏创作者

周二：

- 帮创作者写第一个故事创意
- 用 PlayDrama 做出第一版结构
- 记录卡点

周三：

- 试玩作品
- 记录用户在哪个节点流失
- 整理功能需求

周四：

- 把高频问题交给产品/开发 AI
- 更新模板和教程

周五：

- 发布一条样片或教程
- 总结本周数据
- 更新下周重点

## 创作者内测检查

每个内测创作者至少要完成：

- 一个作品标题
- 三个以上角色
- 四个以上剧情节点
- 五条以上选择分支
- 一个结局
- 一个付费或解锁点
- 一次手机预览试玩

## 运营不要做什么

- 不要承诺已经支持支付。
- 不要承诺已经可以公开上架。
- 不要承诺 AI 视频生成已完成。
- 不要让创作者上传没有授权的真人肖像或版权素材。

## 对外怎么介绍

推荐说法：

```text
PlayDrama Studio 是一个 AI 互动短剧创作工具，帮助创作者把短剧、小说、剧本变成可玩的分支剧情 Demo。
```

不要说：

```text
已经是完整商业平台。
已经能自动生成完整游戏。
已经能替代专业游戏引擎。
```
