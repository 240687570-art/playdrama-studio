# H5 商用内测试跑手册

目标：用线上真实环境跑通“登录、保存、发布、分享、支付宝小额支付、付费结局解锁、数据回传”这一条最小商用闭环。

测试地址：https://playdrama.tokenaicloud.com/studio

## 前置状态

- 数据库：PostgreSQL 已连接。
- 登录：腾讯云短信验证码登录已接入。
- AI：通义千问 `qwen-plus` 已配置。
- 邮件：阿里云邮件推送已配置。
- 支付：支付宝已配置，回调已做签名、金额、币种、AppID 二次校验。
- 分发：H5 链接可用；抖音/微信小程序平台内发布仍等待平台 AppID、业务域名、授权和审核。

## 试跑步骤

1. 打开线上工作台。
   - 访问：https://playdrama.tokenaicloud.com/studio
   - 用真实手机号获取验证码并登录。

2. 保存一个正式测试作品。
   - 确认标题、剧情节点、角色、变量、付费结局文案。
   - 点击“保存”。
   - 预期结果：作品出现在作品库，刷新页面后仍能恢复。

3. 生成发布包。
   - 打开“发布变现”。
   - 执行内容安全扫描。
   - 点击生成或保存发布包。
   - 预期结果：发布包 ID 生成，H5 预览链接可打开。

4. 验证 H5 播放体验。
   - 打开预览链接。
   - 选择至少 3 次剧情选项。
   - 走到付费结局节点。
   - 预期结果：未支付前不能解锁付费结局。

5. 支付宝小额支付。
   - 使用最低可接受金额测试。
   - 完成支付宝支付。
   - 返回 H5 页面。
   - 预期结果：订单变为已支付，付费结局可解锁。

6. 验证数据回传。
   - 回到工作台“发布变现”。
   - 检查播放数据、渠道来源、结局到达事件。
   - 预期结果：能看到访问、选择、结局相关数据。

7. 验证审计记录。
   - 回到“项目总览”。
   - 检查项目活动和最近操作。
   - 预期结果：保存、发布、支付相关操作有记录。

## 通过标准

- 登录验证码能收到，登录后会话可恢复。
- 作品保存后刷新不丢。
- 发布链接能打开。
- 未支付不能看付费结局。
- 支付成功后可以解锁。
- 支付回调无异常，订单状态正确。
- 数据和审计有记录。
- 页面无明显错字、乱码、空白、控制台错误。

## 失败处理

- 收不到验证码：检查腾讯云短信额度、签名、模板、手机号频控。
- 支付后未解锁：检查支付宝回调、订单状态、金额校验、AppID 校验。
- 发布链接打不开：检查 `APP_BASE_URL`、Nginx、服务进程和发布包 ID。
- 数据未回传：检查浏览器请求、analytics 事件、渠道参数和数据库写入。
