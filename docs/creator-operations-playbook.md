# Creator Operations Playbook

## Creator Promise

Creators should be able to make a playable interactive drama faster than they can brief a traditional production team.

## Ideal First Creator

- Already writes short drama, fiction, game stories, or scripts.
- Has an audience or publishing channel.
- Can give feedback quickly.
- Wants to test interactive monetization.

## Onboarding Script

1. Ask for one story idea.
2. Turn it into a 5-10 minute interactive outline.
3. Create three characters.
4. Add at least three choices.
5. Add one hidden ending.
6. Publish H5 preview.
7. Review completion and choice data.

## First Project Checklist

- Strong opening hook
- Clear protagonist goal
- Three meaningful choices
- One clue or puzzle
- At least two endings
- One monetization test point
- Mobile preview checked
- Share link ready

## Creator Success Metrics

- Time to first playable work
- Number of published works
- Repeat publishing rate
- Completion rate
- Paid conversion
- Creator satisfaction

## Weekly Creator Program

Monday:

- Recruit and invite creators.

Tuesday:

- Onboard new creators.

Wednesday:

- Review published works and data.

Thursday:

- Ship product fixes from feedback.

Friday:

- Publish showcase, tutorial, or ranking.
