# PlayDrama Studio PRD

## 1. Product Positioning

PlayDrama Studio is an AI interactive drama game studio for creators. The first product focuses on suspense interactive short dramas that can be generated, edited, published as H5, measured, and monetized.

The goal is not to compete as another AI video generator. The goal is to help creators produce playable, testable, and monetizable interactive drama IP.

## 2. First MVP

MVP theme: AI suspense interactive drama generator.

Core flow:

```text
Idea -> AI story outline -> characters -> story graph -> interactive components -> H5 preview -> publish link -> analytics
```

## 3. Target Users

- Short drama creators who want interactive branches and paid endings.
- Writers who want to turn scripts or novels into playable stories.
- Indie creators who need a fast prototype before investing in video production.
- MCN or content teams testing new IP formats.

## 4. Core Modules

### Creator Workspace

- Project dashboard
- Draft saving
- Version history
- Publish status

### AI Story Generator

- Generate a 5-10 minute interactive script from one idea.
- Generate hook, conflict, choices, clues, endings, and paid unlock points.
- Rewrite by genre, tone, length, or audience.

### Character System

- Character name, role, visual reference, voice, personality, memory, relationship map.
- Consistency rules for future image, video, and dialogue generation.

### Story Engine

- Story nodes
- Choices
- Variables
- Conditions
- Endings
- Inventory and clue state

### Interaction Components

MVP components:

- Dialogue choice
- Clue search
- Password lock
- QTE
- Paid ending gate

### H5 Player

- Mobile-first runtime
- Choice UI
- State tracking
- Ending resolution
- Share link

### Analytics

- Views
- Completion rate
- Choice distribution
- Drop-off node
- Ending rate
- Paid conversion

## 5. Success Criteria

MVP success:

- 100 registered creators
- 30 published works
- 10 public demo works
- 3 works with paid unlock tests
- One playable work can be produced in under 60 minutes
