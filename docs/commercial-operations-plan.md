# Commercial Operations Plan

## Business Goal

Turn PlayDrama Studio into a creator platform for AI interactive drama games, starting with suspense and expanding into romance, horror, fantasy, and short-drama IP.

## Initial Market

Primary beachhead:

- Chinese short drama creators
- Interactive fiction writers
- Indie game prototype creators
- MCN teams testing AI interactive content

Secondary global markets:

- US AI creator community
- Japan visual novel and character fandom
- Korea short drama and romance content
- Southeast Asia mobile H5 entertainment

## Commercial Positioning

PlayDrama Studio helps creators:

- Create playable interactive drama faster.
- Keep characters and story state consistent.
- Publish H5 experiences.
- Test retention and choices.
- Monetize with paid endings, chapters, ads, and templates.

## Product Packaging

### Free

- Limited projects
- Limited AI generation quota
- Watermarked publish link
- Community templates

### Creator

- More projects
- Higher AI quota
- No watermark
- Basic analytics
- Paid ending tests

### Studio

- Team workspace
- Commercial license
- Advanced analytics
- Template marketplace access
- Priority generation queue

### Enterprise

- Private deployment or dedicated workspace
- Custom templates
- Brand/IP safety workflow
- API access
- SLA and support

## Go-To-Market Plan

### Phase 1: Demo Proof

Actions:

- Build 3 public demo works.
- Publish videos showing the creation process.
- Recruit 30 beta creators.
- Collect 10 real scripts or story ideas.

Channels:

- Bilibili
- Xiaohongshu
- Douyin
- WeChat groups
- GitHub
- Product Hunt later for global launch

### Phase 2: Creator Beta

Actions:

- Invite 10 creators into private beta.
- Help each creator publish one work.
- Measure time to first playable work.
- Interview creators after publishing.

Metrics:

- Activation rate
- Published work rate
- Creation time
- Completion rate
- Creator willingness to pay

### Phase 3: Paid Test

Actions:

- Add paid ending or paid chapter flow.
- Test three pricing models.
- Run traffic to 3-5 works.
- Track conversion and refund risk.

Metrics:

- Pay conversion above 3%
- Completion above 35%
- Creator D7 retention above 25%
- Cost per generated publishable minute

### Phase 4: Platform Launch

Actions:

- Launch template marketplace.
- Launch creator profile pages.
- Launch demo gallery.
- Launch affiliate or referral program.
- Publish weekly best interactive drama ranking.

## Operating Workflows

### Creator Acquisition

```text
Find creator -> invite with demo -> onboard -> help create first project -> publish -> review data -> upsell
```

### Content Production

```text
Select genre -> create template -> generate sample work -> polish opening -> add monetization node -> publish demo
```

### Product Feedback

```text
Creator feedback -> tag issue -> classify pain -> prioritize -> implement -> announce update
```

### Release Communication

Every release should include:

- What changed
- Who benefits
- How to use it
- One demo or screenshot
- Next planned improvement

## Revenue Streams

- Monthly subscription
- AI generation credits
- Paid endings and chapters
- Template marketplace commission
- Creator tipping commission
- Enterprise licensing
- IP adaptation services

## Key Risks

- AI generation cost too high
- Video and character consistency weak
- Creators publish low-quality content
- Public server moderation burden
- Copyright or likeness disputes
- Payment and revenue share complexity

## Risk Controls

- Start with image and text-heavy H5 before expensive video generation.
- Use templates and quality checker to raise baseline quality.
- Keep first beta private.
- Require user-owned or licensed materials.
- Add content moderation before public discovery.
- Add payments only after engagement is validated.
